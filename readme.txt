# Simple Text Editor

A simple text editor built with Python and Tkinter. It supports basic text editing features such as opening, saving, and creating new files, as well as text formatting, finding and replacing text, and text highlighting.

## Features

- **File Operations**
  - New (Ctrl+N)
  - Open (Ctrl+O)
  - Save (Ctrl+S)
  - Save As (Ctrl+Shift+S)
  - Exit (Ctrl+Q)

- **Edit Operations**
  - Cut (Ctrl+X)
  - Copy (Ctrl+C)
  - Paste (Ctrl+V)
  - Find (Ctrl+F)
  - Replace (Ctrl+H)
  - Highlight (Ctrl+L)

- **View Operations**
  - Zoom In (Ctrl++)
  - Zoom Out (Ctrl+-)

- **Format Options**
  - Choose Font
  - Choose Font Size

- **About**
  - About Developer
